Crea il database se non esiste inizializzandolo con dati predefiniti. Controlla l'esistenza del file database.json. Se assente, crea la struttura con array vuoti.

Precondition
Il sistema deve avere i permessi di scrittura nella cartella di esecuzione.
Postcondition
Viene creato il file database.json con la struttura base.
Exceptions
IOExceptionSe si verifica un errore durante la scrittura del file.
